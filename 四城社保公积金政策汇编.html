五、通用操作指南（四城通用）

5.1 单位社保开户

  1. 取得营业执照后，系统自动完成社保登记（广东省已实现"多证合一"）；
  2. 登录广东省电子税务局（etax.guangdong.chinatax.gov.cn），以企业身份登录；
  3. 路径：【我要办税】→【税费申报及缴纳】→【社保费管理】；
  4. 首次使用需完成社保费基本信息管理中的单位信息确认与缴费核定；
  5. 绑定三方扣款协议（网签三方协议），确保扣款账户余额充足。

5.2 增员（新员工参保）

  1. 登录广东省电子税务局 → 社保费管理 → 社保增员登记；
  2. 录入新员工信息（姓名、身份证号、缴费工资、参保险种）；
  3. 系统自动校验，校验通过后提交；
  4. 也可通过"粤税通"小程序：微信搜索"粤税通"→社保业务→增员；
  5. 增员后当月需完成社会保险费申报才能扣款。

5.3 减员（员工离职停保）

  1. 登录电子税务局 → 社保费管理 → 社保减员登记；
  2. 查询在册人员，勾选需减员人员，选择减员原因（如"解除劳动合同"）；
  3. 填写实际减员时间 → 校验 → 提交；
  4. 注意：如当月已为该员工申报社保，需先作废申报再减员；
  5. "粤税通"小程序也可办理减员。

5.4 缴费工资（基数）调整

  1. 登录电子税务局 → 社保费管理 → 缴费工资调整；
  2. 选择需调整的员工，录入新的缴费工资；
  3. 系统自动判断是否在基数上下限范围内；
  4. 每年7月为养老保险基数统一调整月，建议在此之前完成年度工资申报；
  5. 灵活就业人员通过"粤税通"【灵活就业人员社保】→【缴费基数调整】。

5.5 公积金操作

  1. 各市公积金中心有独立的网上办事大厅（广州：gjj.gz.gov.cn；珠海：gjj.zhuhai.gov.cn等）；
  2. 年度调整：每年7月在公积金系统完成基数/比例调整申报；
  3. 增员/减员：公积金系统单独操作，不与社保同步；
  4. 灵活就业人员可通过各市公积金微信公众号或网上大厅办理缴存。

5.6 跨市转移

  1. 省内转移：广东省内四城之间社保关系转移已简化，新单位直接增员即可；
  2. 养老保险：省内无需办理转移手续，退休时统一归集；跨省需通过"掌上12333"APP或国家社保平台申请；
  3. 医疗保险：通过"粤医保"小程序办理省内转移；跨省需申请；
  4. 公积金：通过"全国住房公积金"小程序办理异地转移接续。

六、资深社保专员补充建议

🔴 1. 广州医保费率恢复——预算冲击

2026年1月起广州医保单位费率从4.5%恢复至6%（含生育从5.35%升至6.85%），一名最低基数员工企业年增成本约1,121元。建议：

  • 提前与财务沟通2026年社保预算上调
  • 关注是否出台新阶段性降费政策
  • 评估灵活用工替代方案

🟠 2. 佛山医保基数"洼地效应"

佛山医保基数上限仅9,660元，远低于养老基数的27,549元。这意味着：

  • 月薪2万的员工在佛山缴医保成本远低于其他三城
  • 高薪人群的医保待遇可能受影响（缴费基数封顶低）
  • 跨市招聘时可作为成本优势宣传

🟡 3. 四城最低工资标准与失业金

广州最低工资2,500元，珠海/东莞/佛山均为2,080元。失业保险金=最低工资×90%：

  • 广州：2,250元/月
  • 珠海/东莞/佛山：1,872元/月
  • 广州高17%——跨市用工的隐性成本差异

🟢 4. 医保退休年限差异

  • 广州：2014年后首次参保需累计15年+本市满10年
  • 珠海：到2029年统一为25年+本市满10年
  • 东莞：按广东省统一规定逐步调整
  • 佛山：按省规定，关注过渡期安排

员工临近退休时跨市流动需特别注意医保年限能否满足退休地要求。

🔵 5. 公积金缴存策略

四城公积金均为5%-12%区间，但：

  • 东莞灵活就业最低10%，比单位职工5%高出一倍
  • 佛山要求个人比例≥单位比例（不能单位12%个人5%）
  • 广州基数上限最高（41,697元），高薪人群公积金账户积累最多
  • 公积金贷款额度与账户余额、缴存年限挂钩，提前规划

🟣 6. 社保合规红线提醒

  • 禁止按最低基数全员参保：广东省已推行"社保费征缴数据比对"，税务局有个人所得税工资申报数据，低报将触发稽查
  • 禁止选择性参保：五险必须全部参加（特定人员除外）
  • 禁止公积金不缴：公积金具有强制性，违规罚款1-5万元
  • 试用期必须参保：建立劳动关系30日内办理

🟤 8. 灵活就业参保策略
