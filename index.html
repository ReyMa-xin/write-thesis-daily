四、资深社保专员实操建议

1. 广州医保费率恢复

2026年1月单位费率从4.5%恢复至6%（含生育5.35%→6.85%），最低基数员工企业年增≈1,121元/人。建议更新2026年社保预算。

2. 佛山医保基数洼地

医保基数上限仅9,660元，远低于养老基数27,549元。高薪员工医保成本约为其他三城的1/3。

3. 最低工资与失业金差异

广州最低工资2,500元，失业金2,250元/月；其他三城2,080元，失业金1,872元/月。广州高出20%。

4. 医保退休年限

  • 广州：2014年后参保需累计15年+本市满10年
  • 珠海：至2029年逐步统一为25年+本市满10年
  • 东莞/佛山：按省规定逐步调整

5. 公积金策略

  • 东莞灵活就业最低10%（单位职工的2倍）
  • 佛山：个人比例必须 ≥ 单位比例
  • 广州基数上限最高（41,697元）

6. 合规红线

  • 禁止全员按最低基数参保（税务已比对个税数据）
  • 公积金具有强制性，违规罚款1-5万元
  • 试用期30日内必须参保

7. 通用操作渠道

增减员/基数调整 → 广东省电子税务局 或 "粤税通"小程序；公积金 → 各市公积金中心网上办事大厅。社保咨询热线：12333 / 12366。

8. 灵活就业最优策略

  • 珠海：选单建统筹（2.5%）省成本
  • 东莞：选统账结合无个账折中
  • 佛山：医保基数下限仅4,669元

五、通用操作指南（四城通用）

5.1 单位社保开户

  1. 取得营业执照后系统自动完成社保登记（广东省"多证合一"）；
  2. 登录广东省电子税务局（etax.guangdong.chinatax.gov.cn）；
  3. 路径：【我要办税】→【税费申报及缴纳】→【社保费管理】；
  4. 完成社保费基本信息管理中的单位信息确认与缴费核定；
  5. 绑定三方扣款协议，确保扣款账户余额充足。

5.2 增员（新员工参保）

  1. 电子税务局 → 社保费管理 → 社保增员登记；
  2. 录入新员工信息（姓名、身份证号、缴费工资、参保险种）；
  3. 系统自动校验，也可通过"粤税通"小程序操作；
  4. 增员后当月需完成社会保险费申报才能扣款。

5.3 减员（员工离职停保）

  1. 电子税务局 → 社保费管理 → 社保减员登记；
  2. 勾选需减员人员，选择减员原因，填写实际减员时间；
  3. 如当月已申报社保，需先作废申报再减员。

5.4 缴费工资调整

  1. 电子税务局 → 社保费管理 → 缴费工资调整；
  2. 系统自动判断是否在基数上下限范围内；
  3. 每年7月为养老保险基数统一调整月，建议在此之前完成年度工资申报。

5.5 公积金操作

  1. 各市公积金中心有独立网上办事大厅；
  2. 年度调整：每年7月在公积金系统完成基数/比例调整；
  3. 增员/减员：公积金系统单独操作，不与社保同步。

5.6 跨市转移

  1. 省内转移：四城之间已简化，新单位直接增员即可；
  2. 养老保险：省内无需办理转移，退休时统一归集；
  3. 医疗保险：通过"粤医保"小程序办理；
  4. 公积金：通过"全国住房公积金"小程序办理。

六、资深社保专员补充建议

🔴 1. 广州医保费率恢复——预算冲击

2026年1月起广州医保单位费率从4.5%恢复至6%，最低基数员工企业年增成本约1,121元/人。建议提前与财务沟通2026年社保预算，关注是否出台新降费政策。

🟠 2. 佛山医保基数"洼地效应"

佛山医保基数上限仅9,660元，远低于养老基数的27,549元。高薪员工的医保成本远低于其他三城，跨市招聘时可作为成本优势宣传。

🟡 3. 最低工资与失业金差异

广州最低工资2,500元，失业金2,250元/月；其他三城2,080元，失业金1,872元/月。广州高出17%，是跨市用工的隐性成本。

🟢 4. 医保退休年限差异

  • 广州：2014年后参保需累计15年+本市满10年
  • 珠海：至2029年统一为25年+本市满10年
  • 东莞/佛山：按省规定逐步调整

🔵 5. 公积金缴存策略

  • 东莞灵活就业最低10%，比单位职工5%高出一倍
  • 佛山：个人比例≥单位比例
  • 广州基数上限最高（41,697元），高薪人群公积金贷款额度优势最大

🟣 6. 社保合规红线

  • 禁止按最低基数全员参保（税务已比对个税数据）
  • 公积金具有强制性，违规罚款1-5万元
  • 试用期30日内必须参保
